# Documentacion

Usa esta carpeta para documentacion base del proyecto que no sea contractual ni correo. Esta es una carpeta de **material fuente** fuera de `.github\`.

| Subcarpeta | Uso |
|------------|-----|
| `Esquemas\` | Planos, diagramas, P&ID, esquemas electricos o funcionales |
| `Manuales\` | Manuales, datasheets, fichas tecnicas y referencias |
| `Programas\` | Backups, configuraciones, listados de parametros o software entregado |

Anade nuevas subcarpetas solo cuando aporten una clasificacion estable al proyecto.

Cuando un documento de esta carpeta cambie alcance, requisitos, interfaces o riesgos, actualiza `.github\context\` y `.github\planning\`.
