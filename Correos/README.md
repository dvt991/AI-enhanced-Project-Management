# Correos

Usa esta carpeta para correos, hilos exportados, mensajes `.msg` o `.eml`, y evidencia de coordinacion. Esta es una de las carpetas de **material fuente** que deben permanecer fuera de `.github\`.

## Reglas

- Guarda solo mensajes relevantes para el proyecto.
- Usa nombres legibles como `YYYY-MM-DD_remitente_asunto.ext`.
- Cuando un correo genere acciones o decisiones, actualiza `.github\planning\backlog.md`, `.github\tracking\action-tracker.csv` o `.github\decisions\`.
- Si el correo cambia alcance, riesgos, planning o coste, refleja ese impacto tambien en `.github\context\` y `.github\tracking\`.
- Cuando la plantilla se reutilice, elimina cualquier correo del proyecto anterior.
