# Tecnico

Usa esta carpeta para documentacion de ejecucion, soporte de campo, pruebas, punch lists, checklists y seguimiento tecnico del proyecto. Esta es una carpeta de **material fuente** fuera de `.github\`.

La subcarpeta `Seguimiento_Materiales\` queda reservada para albaranes, POD, custodias, recepciones y trazabilidad logistica.

Cuando aparezcan hallazgos tecnicos, avances de sitio o incidencias, actualiza `.github\context\`, `.github\planning\` y `.github\tracking\`.
