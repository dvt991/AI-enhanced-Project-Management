# Oferta_Pedido

Usa esta carpeta como baseline contractual y comercial del proyecto. Esta es una carpeta de **material fuente** fuera de `.github\`.

## Contenido tipico

- oferta enviada o version final acordada
- pedido, PO o documento de aceptacion
- contrato, anexos y condiciones aplicables
- cambios, claims o aprobaciones comerciales
- evidencias de firma o aceptacion

La subcarpeta `Proveedores\` queda reservada para ofertas y soporte de subcontratas o vendors.

Cuando la baseline cambie, actualiza `.github\context\`, `.github\planning\` y `.github\tracking\`.
